# pomodoro

> The F2E 第二屆 - 前端精神修煉時光屋/第一關/蕃茄鐘

#### Build Setup

``` bash
# install dependencies
yarn install

# serve with hot reload at localhost:9080
yarn run dev

# build electron application for production
yarn run build


# lint all JS/Vue component files in `src/`
yarn run lint

```
